# RosaFiesta

RosaFiesta is a comprehensive event management platform built as a monorepo.

## Architecture

### Backend
-   **Language**: Go (Golang)
-   **Database**: PostgreSQL
-   **Framework**: Chi Router, Standard Library
-   **Location**: [`/Backend`](./Backend)

### Mobile App (Frontend)
-   **Framework**: Flutter (Dart)
-   **Platforms**: Android, iOS, Web, macOS, Windows, Linux
-   **Location**: [`/MobileApp`](./MobileApp)

## Getting Started

### Backend
Navigate to `/Backend` and run:
```bash
go run cmd/main/main.go
```

### Mobile App
Navigate to `/MobileApp` and run:
```bash
flutter run
```
